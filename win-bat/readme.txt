Windows Use
-------------------------------
After unzipping this directory:
* use 'compile.bat' to generate the *.class files
* use 'make.bat' to package them into an executable jar
* use 'run.bat' to launch the jar
-- these directions assume that the java/bin directory is added to your classpath
